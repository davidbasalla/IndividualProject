1.) Medical Image Volume Example Files

For testing of the basic viewing features of ScanView. Download this file to your computer, create a new Layer in ScanView, click on the Open Button in the new Layer and select this file.

Brain (.nii)
Thorax (.nrrd)

2.) Annotation Example Files

For testing the Annotation feature of ScanView. Download this file to your computer, in ScanView (once you have loaded a medical image data file), go to the Annotations tab and click 'Load' or 'Import' and select this file. Alternatively you can just create a new Annotation by clicking on 'New'.

Brain Annotations (.json)

3.) Labelmap Example Files

For testing the Labelmap feature of ScanView. Download this file to your computer, in ScanView (once you have loaded a medical image data file), go to the Labelmap tab and click the 'Open' button and select this file.

Brain Labelmap (.nii)
